# lunar-lib

This repository contains official Lunar collector and policy plugins.

Repo structure:

- `ai-context/`: AI context necessary for building new plugins.
- `collectors/`: Contains collector plugins.
- `policies/`: Contains policy plugins.
- `inc/collectors/`: Contains incubation collector plugins. These have been AI generated and only minimally reviewed.
- `inc/policies/`: Contains incubation policy plugins. These have been AI generated and only minimally reviewed.
