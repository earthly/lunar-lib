# Advanced Packs

**Coming soon.**

Advanced packs are for specific use cases or require significant configuration to be useful.

## Planned Packs

Advanced packs will cover specialized workflows and tools that need meaningful setup:

- **AI Governance** — AI assistant usage tracking, instruction file standards (`ANTHROPIC_API_KEY`)
- **Custom SAST Rules** — AST-grep with custom YAML rule definitions
- **OpenTelemetry** — CI observability with OTel endpoint configuration
- **GitOps** — ArgoCD/Flux deployment pattern detection
- **Supply Chain Signing** — Cosign image signing verification
- **Code Complexity** — McCabe/cognitive complexity thresholds
- **CI Platform-Specific** — Jenkins, CircleCI, Azure DevOps, Buildkite

Each pack will follow the same format as the [Starter](../starter/) packs — a `lunar-config.yml` with curated policies and enforcement levels.
